ВСЁ ЧТО НУЖНО ДЕЛАТЬ:

1️⃣ Установить зависимости:
   pip install opencv-python PyQt5 numpy requests

2️⃣ Запустить генератор конфига:
   python generate_mediamtx_config.py

3️⃣ Запустить MediaMTX:
   mediamtx mediamtx.yml

4️⃣ Запустить приложение:
   python professional_client.py

5️⃣ Импортировать камеры:
   File → Import from MediaMTX → Import

6️⃣ ГОТОВО! Все 130 камер видны и работают! 🎉
