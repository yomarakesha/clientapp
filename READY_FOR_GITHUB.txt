🚀 **ПЕРВЫЙ PUSH НА GITHUB - 3 КОМАНДЫ:**

1. На GitHub создайте новый репозиторий:
   https://github.com/new
   
   Назовите: mediamtx-professional-client
   Выберите: Public

2. В PowerShell выполните:

   cd "C:\Users\Windows 11 Pro\Desktop\clientapp"
   git init
   git add .
   git commit -m "Initial commit: Professional MediaMTX client v1.0

   - Support for 130+ cameras
   - Auto-import from MediaMTX API
   - Professional dark theme
   - Multi-view layouts
   - Video recording and screenshots
   - Event logging system
   - Complete Russian documentation"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/mediamtx-professional-client.git
   git push -u origin main

3. Готово! Проверьте на GitHub:
   https://github.com/YOUR-USERNAME/mediamtx-professional-client

   ✅ Все файлы загружены
   ✅ README красиво отображается
   ✅ История коммитов видна
