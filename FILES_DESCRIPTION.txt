🔧 ВСПОМОГАТЕЛЬНЫЕ СКРИПТЫ:

  🔧 generate_mediamtx_config.py (150+ строк)
     ━━━━━━━━━━━━━━━━━━━━━━━━━━━━
     Генератор конфига для MediaMTX
     
     Делает:
     • Создаёт mediamtx.yml для 130 камер
     • Поддерживает Hikvision, Dahua, Uniview
     • Генерирует cameras.json для импорта
     
     Запуск: python generate_mediamtx_config.py
     
     Интерактивный - выбираете тип NVR и количество камер

📁 КОНФИГИ (создаются/редактируются приложением):

  📋 cameras.json
     ━━━━━━━━━━━━━━
     Список всех камер в JSON формате
     
     Пример:
     [
       {
         "name": "Вход 1",
         "url": "rtsp://127.0.0.1:8554/camera_001",
         "group": "Входы",
         "source": "mediamtx"
       },
       ...
     ]
     
     Как получить:
     • Автоматически через импорт
     • Ручное редактирование
     • Экспорт из приложения

  ⚙️ config.json
     ━━━━━━━━━━━━
     Параметры приложения
     
     Содержит:
     • URL MediaMTX сервера
     • Параметры записи (FPS, битрейт)
     • Директория для записей
     • Ваши предпочтения
     
     Редактируется в: Tools → Settings

  📊 events.json
     ━━━━━━━━━━━━
     Полная история событий (журнал логов)
     
     Записывает:
     • Подключение/отключение камер
     • Запуск/остановка записи
     • Скриншоты
     • Добавление/удаление камер
     • Ошибки и предупреждения
     
     Просмотр: Tools → Event Log

✅ ПРЕДПОЛЕТНАЯ ПРОВЕРКА:

Перед началом убедитесь что у вас есть:

□ Python 3.8+ (проверка: python --version)
□ NVR с 130 камерами (включены, выдают потоки)
□ Знание типа вашего NVR (Hikvision/Dahua/другой)
□ IP адрес NVR
□ Интернет для скачивания MediaMTX
□ ПК для запуска MediaMTX и приложения
□ SSD для хранения видео (если нужна запись)

Если всё есть ✅ - готовы к началу!

🚀 БЫСТРЫЙ СТАРТ (из этой папки):

Откройте PowerShell/Terminal в папке clientapp/ и выполните:

1. Установка зависимостей:
   pip install opencv-python PyQt5 numpy requests

2. Генерация конфига MediaMTX:
   python generate_mediamtx_config.py
   (выбираете тип NVR, вводите IP и кол-во камер)

3. Запуск MediaMTX:
   mediamtx mediamtx.yml
   (консоль выведет: "camera_001 ready", "camera_002 ready", ...)

4. В новом окне PowerShell запуск приложения:
   python professional_client.py

5. В приложении:
   File → Import from MediaMTX
   URL: http://127.0.0.1:9997
   Import
   
   ВСЕ 130 КАМЕР ИМПОРТИРОВАНЫ! 🎉

📊 ТЕСТЫ:

Проверьте что всё работает:

1. Браузер → http://127.0.0.1:9997/list
   Должны видеть JSON со всеми потоками ✓

2. VLC → Открыть сетевой поток
   rtsp://127.0.0.1:8554/camera_001
   Должны видеть видео ✓

3. Приложение → Import from MediaMTX
   Должны импортировать 130 ✓

4. Выберите камеру → Play
   Должны видеть прямую трансляцию ✓

Если все ✓ - всё работает отлично!
