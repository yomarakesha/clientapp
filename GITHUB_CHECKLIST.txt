🚀 **ПЕРВЫЙ PUSH НА GITHUB** (пошагово):

1️⃣  СОЗДАТЬ РЕПОЗИТОРИЙ НА GITHUB:
    ✓ Откройте: https://github.com/new
    ✓ Name: mediamtx-professional-client
    ✓ Description: Professional CCTV client for 130+ cameras
    ✓ Public
    ✓ Создайте репозиторий

2️⃣  ИНИЦИАЛИЗИРОВАТЬ GIT ЛОКАЛЬНО:
    
    # Откройте PowerShell в папке проекта:
    cd "C:\Users\Windows 11 Pro\Desktop\clientapp"
    
    # Инициализировать git:
    git init
    
    # Добавить все файлы:
    git add .
    
    # Первый коммит:
    git commit -m "Initial commit: Professional MediaMTX client v1.0
    
    - Support for 130+ cameras
    - Auto-import from MediaMTX API
    - Professional dark theme UI
    - Multi-view layouts
    - Video recording and screenshots
    - Real-time video editing
    - Event logging system
    - Full documentation"
    
    # Переименовать в main:
    git branch -M main
    
    # Добавить GitHub репозиторий:
    git remote add origin https://github.com/YOUR-USERNAME/mediamtx-professional-client.git
    
    # Первый push:
    git push -u origin main

3️⃣  ПРОВЕРИТЬ НА GITHUB:
    ✓ Откройте: https://github.com/YOUR-USERNAME/mediamtx-professional-client
    ✓ Видны все файлы
    ✓ README.md отображается красиво
    ✓ История коммитов видна

📝 **ОБНОВЛЕНИЯ ПОСЛЕ PUSH** (для будущего):

    # Когда будете добавлять новые функции:
    git add .
    git commit -m "Add: [описание новой функции]"
    git push
    
    # Хорошие префиксы для коммитов:
    - Add:     новая функция
    - Fix:     исправление ошибки
    - Update:  обновление документации
    - Improve: улучшение производительности
    - Refactor: переделка кода
    - Remove:  удаление
